Enunciado: Crea dos variables, `$nombre` y `$edad`, y luego imprime "Mi nombre es [NOMBRE] y tengo [EDAD] años". 

<?php
$nombre = "Juan";
$edad = 25;
echo "Mi nombre es $nombre y tengo $edad años.";
?>
Enunciado: Dados dos números, `$a = 10` y `$b = 20`, imprime la suma, resta, multiplicación y división de estos números. 

<?php
$a = 10;
$b = 20;
echo "Suma: " . ($a + $b) . "<br>";
echo "Resta: " . ($a - $b) . "<br>";
echo "Multiplicación: " . ($a * $b) . "<br>";
echo "División: " . ($a / $b);
?>
Enunciado: Dado un número, verifica si es par o impar. 

<?php
$numero = 15;
if ($numero % 2 == 0) {
    echo "El número es par.";
} else {
    echo "El número es impar.";
}
?>
Enunciado: Imprime los números del 1 al 10 usando un ciclo `for`. 

<?php
for ($i = 1; $i <= 10; $i++) {
    echo $i . "<br>";
}
?>
Enunciado: Crea un array con los días de la semana y luego imprime cada día usando un ciclo `foreach`. 

<?php
$dias = ["Lunes", "Martes", "Miércoles", "Jueves", "Viernes", "Sábado", "Domingo"];
foreach ($dias as $dia) {
    echo $dia . "<br>";
}
?>
Enunciado: Escribe una función que tome dos números y devuelva su suma. 

<?php
function suma($a, $b) {
    return $a + $b;
}
echo suma(5, 10);  // Salida: 15
?>

Enunciado: Establece una cookie que almacene el nombre del usuario y luego recupérala para mostrar un mensaje de bienvenida. 

<?php
// Establecer cookie
setcookie("nombre_usuario", "Juan", time() + 3600, "/");
// Recuperar cookie y mostrar mensaje
if(isset($_COOKIE["nombre_usuario"])) {
    echo "¡Hola, " . $_COOKIE["nombre_usuario"] . "!";
}
?>
Enunciado: Inicia una sesión y almacena una variable de sesión con un mensaje. Luego, recupera esa variable y muéstrala. 

<?php
// Iniciar sesión
session_start();
// Establecer variable de sesión
$_SESSION["mensaje"] = "Aprender PHP es divertido!";
// Recuperar y mostrar variable de sesión
if(isset($_SESSION["mensaje"])) {
    echo $_SESSION["mensaje"];
}
?>
